Detailed Description

pclVoxelTest.cpp consists of 3 unit test cases that test the pclVoxel class.

TEST(pclVoxelTest, setLeafValue) will test the setLeafSize() and getLeafSize() method. TEST(pclVoxelTest, pointCloudDownSampleing) will test the filterProcess() method. TEST(pclVoxelTest, setpclCloud) will test the getInputCloud() and setInputCloud() method.
